# Vue2 + webpack2 + Yarn 基础项目框架

本项目使用yarn替换传统的npm对包进行管理，yarn可以默认对依赖包锁定初次安装时对版本，解决传统方式下js包默认升级造成对依赖包版本不兼容对问题

## 运行步骤

``` bash
# 项目推荐使用yarn进行包管理，需要预先安装全局yarn
npm install yarn -g

# 安装依赖
yarn install

# 开发环境启动
npm run dev

# 线上环境代码打包
npm run build

```
